## Installation

```
\#>git clone https://github.com/TEdit/terraria-map-editor-web
\#>npm install
(if you can fix unexpected config errors yourself:
&nbsp;&nbsp;&nbsp;&nbsp;\#>npm audit fix
&nbsp;&nbsp;&nbsp;&nbsp;\#>npm update
)
\#>git submodule init
\#>git submodule update
\#>npm start
```
